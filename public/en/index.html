Find answers to common questions about ULT AI Translator for game localization, movie subtitles, and content translation

Our free trial gives you full access to all ULT AI Translator features for 7 days. No credit card is required to start. Simply download the software, create an account, and start translating immediately. You can cancel anytime before the trial ends without any charges.

ULT specializes in multimedia content translation, supporting JSON, TXT, LRC (lyrics), SRT (subtitles), XML, and HTML formats. For game localization, we handle UI text, dialogue scripts, and configuration files. For movies, we support subtitle files while preserving timing and formatting.

ULT uses advanced algorithms to automatically identify content that needs translation. Basic detection uses script features and word frequency for speed, while precise detection uses complex algorithms to minimize false positives. This saves tokens by filtering out untranslatable content like code, numbers, or already translated text.

Yes! ULT supports customizable system prompts with {source_lan} and {target_lan} placeholders. You can create different prompts for game dialogue (casual, formal), movie subtitles (concise, descriptive), or technical documentation (precise, user-friendly). This ensures consistent translation style across your projects.

ULT's intelligent batch system groups sentences (configurable from 1-1000 per request) for efficient processing. For game localization, this helps maintain context across dialogue sequences. For movie subtitles, it preserves timing relationships. If batch requests fail, ULT automatically falls back to line-by-line translation to ensure no content is lost.

ULT requires Windows 10/11 or macOS 10.15+. For optimal performance with large game scripts or movie subtitle files, we recommend 16GB RAM and an SSD drive. The software automatically adjusts batch sizes and processing based on your hardware capabilities.

ULT's project management system remembers your cursor position, language settings, and API configurations for each project. This is essential for game developers working on multiple titles or filmmakers handling different movie projects. You can easily switch between projects and continue exactly where you left off.
